## 3 propiedades CSS utilizadas
**1. display: flex;**  
Sirve para acomodar los elementos en una fila o columna. En esta tarea lo usé para que el menú aparezca en una sola línea.

**2. justify-content: center;**  
Sirve para centrar elementos horizontalmente. Lo usé para que el menú quedara centrado en la página.

**3. text-decoration: none;**  
Sirve para quitar el subrayado de los enlaces. Lo usé para que el menú se viera más limpio.
